# StandStock - Proyecto de Portafolio de Título

**Sistema Integral de Gestión de Inventarios en Tiempo Real para Stands de Centros Comerciales**

## Descripción
StandStock es una solución multiplataforma diseñada específicamente para el retail de micro-formato (stands en malls). Permite el control de stock, alertas de reposición y registro de movimientos en tiempo real mediante una app móvil (Flutter) y un panel web de administración (React), con backend en Supabase.

## Tecnologías Principales
- **Mobile**: Flutter (Dart) - Android e iOS
- **Web Admin**: React + Vite + Tailwind
- **Backend y BD**: Supabase (PostgreSQL + Realtime + Auth)
- **Seguridad**: JWT + Row Level Security (RLS)
